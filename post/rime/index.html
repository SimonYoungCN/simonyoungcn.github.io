Rime × 98 五笔

之前几次想在rime里搞小鹤音形输入法效果都不尽人意，再加上rime自带的皮肤并不算好看，而且当时配置的时候各种出问题，于是就气急败坏地弃坑了。我最近开始学习五笔，一直没有找到心仪的输入法，没想到……

我抱着试一试的心态又安装了rime居然成功了。🤨

前言

最先在选择五笔的版本上就纠结了半天。一会觉得86字根拆得逆天98看起顺眼，一会又觉得98字根爆多86简洁明了，其间还夹杂着自己是为什么要学习五笔的灵魂拷问。终于敲定了98的方案，却三天打鱼，两天晒网，中途放弃了好几次。

我一开始用的是Windows自带的输入法 + WubiLex，后来觉得没有多少自定义的空间，也不太会用那个高科技软件，然后就换成了冰凌输入法。我很喜欢它的皮肤，但是它的词库总是怪怪的。无论是自带的还是从官网上下载的至善词库这个输入法总是要把不常用的候选项放在前面，于是风评还算可以的就还剩一个，那就是——rime。

开始操作

安装

这步应该不用多说，直接去官网下载安装包。

配置

输入方案

首先要解决的问题就是选择一个合适的方案。98五笔虽然资源比起86来说不算多，但用户们都非常团结，一小伙人整合出了一个资源库，里面的资料挺全，从学习资料到各种软件都有（不过个人感觉里面的五笔软件挺简陋的）。我用了软件内置的方案，据说采用的也是经典的98至善词库，方案在GitHub上也有。全clone下来塞进用户文件夹里就大功告成了。

候选框

我不太喜欢rime默认的竖排方案，也不想把打出的字符显示在候选栏而是想让它直接弄到文本框里。于是我二话不说直接打开weasel.custom.yaml然后在patch:下cv如下代码：

1
2

  "style/horizontal": true
  "style/inline_preedit": true

瞬间就变成了我想要的样子。1之前久攻不下，现在一气呵成，爽翻了。

美化

默认的rime皮肤都不太好看，于是我又去找了点皮肤。

找到了一个使用98五笔而且写了很多皮肤的大佬2，选了其中之一方案cv到weasel.yaml最下方。然后就可以去设置里面选了。字体我改成了更纱黑体，是我之前用Linux的输入法字体。

1
2

font_face: "Sarasa UI SC"
label_font_face: "Sarasa UI SC"

暂时先写这么多，后面遇到了问题也好解决。

更新：我还是心心念念着 fcitx5-material-color 的青色皮肤，于是我便自己改了一个，贴在这里记录一下。
